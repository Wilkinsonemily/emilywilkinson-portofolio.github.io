SnackSwap

Healthier Choices, One Swap at a Time

A nutrition-focused web application that helps users find healthier alternatives to packaged foods by viewing nutrition details and recommending smarter swaps based on calories, sugar, and fat.

Project Type

Web Application

Role

Full Stack Developer

View Project Details

Project Overview

SnackSwap is designed to make nutrition information easier to understand and help users make healthier decisions when choosing packaged snacks. Users can search products and view detailed nutrition values such as calories, sugar, fat, protein, sodium, and more.

SnackSwap also supports SDG 3 — Good Health and Well-Being by encouraging nutrition awareness and promoting healthier eating habits through better alternatives (healthy swaps).

Key Features

  • Food Search: Search products by name or product code.
  • Nutrition Details: View calories, sugar, fat, protein, sodium, and more.
  • Healthy Categories: Browse items by categories (low sugar, high protein, high fiber, etc.).
  • Healthy Swap: Get healthier alternatives for selected foods.
  • Admin Dashboard: Manage food and nutrition data.
