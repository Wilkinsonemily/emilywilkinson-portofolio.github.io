A 3D interactive dark fantasy scene built with Three.js, featuring a Dark Warrior, a magical spell circle, and a giant battle hamster in a WebGL environment with lighting, textures, animation, and user interaction.

Project Overview

OVerlord is a 3D computer graphics project built using the Three.js library. The project showcases an interactive dark fantasy scene featuring a Dark Warrior and a giant battle hamster inside a 3D environment.

Developed as part of a Computer Graphics course, this project focuses on 3D scene construction, object modeling, lighting techniques, texture mapping, animation, and user interaction using WebGL and JavaScript.

Key Features

  • Interactive 3D Fantasy Scene: Explore and orbit around the environment in real time.
  • Dark Warrior Controls: Movement and rotation control system for character navigation.
  • Magical Spell Circle: Toggle spell effect with dynamic lighting.
  • Hamster Model (Primitives): Hamster built using geometric primitives.
  • Raycasting Interaction: Click interaction to toggle hamster face texture.
  • Skybox via Cube Mapping: Immersive background with cube textures.
  • 3D Text Rendering: TextGeometry for 3D typography inside the scene.
  • Lighting & Shadows: Multiple light sources with shadow support.

User Controls

  • W / S – Move forward / backward
  • A / D – Move left / right
  • Q / E – Rotate left / right
  • Space – Toggle spell effect
  • Mouse Click – Toggle hamster face texture
  • Mouse Drag – Orbit camera
